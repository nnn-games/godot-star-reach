# Allen Mask NPC 캐릭터 레퍼런스

> **게임**: Star Reach
> **캐릭터**: 앨런 마스크 (Allen Mask) — 우주 기업 "Star Reach" 창립자 겸 CEO
> **모티브**: 일론 머스크 (Elon Musk) 패러디
> **톤**: 아는 사람은 킥킥거리고, 모르는 사람도 웃긴 캐릭터

---

## 1. 캐릭터 핵심 성격

| 키워드 | 설명 |
|--------|------|
| **낙관적 몽상가** | 불가능해 보이는 목표를 당연하다는 듯 말함 |
| **어색한 유머** | 대단한 내용을 발표하면서 뜬금없는 농담을 끼워넣음 |
| **불굴의 의지** | 실패해도 절대 포기 안 함. 실패를 긍정으로 전환하는 능력 |
| **덕후 기질** | 게임, SF, 밈을 사랑하는 너드. 순수한 열정 |
| **과대 약속가** | 항상 비현실적으로 짧은 일정을 제시함 (결국 늦게라도 해냄) |
| **멀티태스커** | 동시에 여러 역할을 맡으며 혼돈을 유발 |

## 3. 핵심 에피소드 (스토리텔링용)

### 3.1 Falcon 1: 세 번의 실패, 네 번째 성공

게임 내 앨런 마스크의 핵심 서사로 활용 가능한 실화 기반 스토리.

| 회차 | 실제 사건 (2006~2008) | 게임 활용 |
|------|----------------------|----------|
| 1차 실패 | 발사 25초 만에 엔진 화재. 연료 라인 부식 | "첫 번째니까. 배울 게 많아." |
| 2차 실패 | 2단 로켓 진동으로 궤도 진입 실패 | "거의 성공했어! 거의!" |
| 3차 실패 | 1단-2단 분리 시 충돌. 자금 거의 바닥 | "3번 실패했다고? 나도 그랬어. 그때가 가장 중요한 순간이야." |
| 4차 성공 | 3차 실패 7주 후 남은 부품으로 성공. 민간 최초 궤도 진입 | "7번 터져도, 42번 터져도 상관없어. 다음 한 번이 모든 걸 바꿀 수 있으니까." |

> 4차 성공 직후 NASA와 16억 달러 계약 체결 → 회사 생존 확정.
> 직원 전체 이메일: "우리가 몇 번 넘어졌는지는 중요하지 않다. 중요한 건 다시 일어나는 것이다."

### 3.2 Tesla를 우주로 (Falcon Heavy, 2018)

- 세계 최강 로켓 Falcon Heavy 첫 발사
- 페이로드: 자신의 빨간 Tesla Roadster + "Starman" 마네킹
- 대시보드에 "DON'T PANIC" 문구 (은하수를 여행하는 히치하이커 오마주)
- 데이비드 보위 "Space Oddity" BGM 설정
- 두 부스터 동시 착륙 → SF 영화 같다는 반응
- **게임 활용**: 앨런 마스크가 자기 자동차를 로켓에 태워 보내는 이벤트 + "당황하지 마!" 대사

### 3.3 드론십 착륙 (2016)

- 바다 위 무인 바지선에 로켓 착륙 성공
- 드론십 이름: "Of Course I Still Love You" (SF 소설에서 따옴)
- 그 전까지 해상 착륙 수차례 실패 (폭발 모음 영상이 밈)
- **게임 활용**: 착륙 패드에 이상한 이름 붙이는 앨런 마스크 특유의 작명 센스

### 3.4 Mechazilla 부스터 캐치 (2024)

- 발사대의 거대한 기계 팔로 로켓 부스터를 공중 포획
- 전 세계 수백만 명 생중계 시청
- **게임 활용**: 거대 로봇 팔 연출 + 앨런 마스크 환호

### 3.5 SpaceX 탄생 비화

- 러시아에서 ICBM 개조 로켓을 사려고 세 번 방문 → 가격 과다로 거절
- 귀국 비행기 안에서 스프레드시트로 로켓 제작 비용 계산 → "직접 만들자!"
- **게임 활용**: "계획이 안 됐을 때가 최고의 기회야!" 대사

## 4. 러닝개그 패턴 (반복 유머)

### 4.1 "앨런 타임" (Elon Time 패러디)
항상 일정을 과소평가하는 습관.
```
앨런 마스크: "이 업그레이드? 5분이면 완성이야!"
(시스템 메시지): 예상 소요 시간: 2시간
앨런 마스크: "...우주적 스케일로 5분이야."
```

### 4.2 이상한 작명 센스
모든 것에 독특한 이름을 붙이는 습관. (실제: 드론십 "Of Course I Still Love You", 제품 "Not a Flamethrower", 자녀 "X Æ A-12")
```
앨런 마스크: "이 발사대 이름은 '물론 아직도 널 사랑해'로 정했어."
플레이어: "...왜요?"
앨런 마스크: "좋은 질문이야! 다음 질문!"
```

### 4.3 실패 긍정 변환
어떤 폭발/실패도 긍정적으로 해석하는 능력.
```
[로켓 폭발 시]
앨런 마스크: "폭발이 아니야! 계획에 없던 불꽃쇼야!"
앨런 마스크: "멋진 데이터를 수집했어! 다음엔 분명 더 멋지게 터질 거야... 아니, 성공할 거야!"
```

### 4.4 갑작스러운 1줄 발표
맥락 없이 짧은 한마디만 하고 사라지는 패턴. (실제: 트위터에 "Mars", "Soon", "Interesting..." 등 맥락 없는 1단어 트윗)
```
앨런 마스크: "중대 발표가 있어!"
(긴장되는 브금)
앨런 마스크: "...곧."
(사라짐)
```

### 4.5 "물리법칙 체크"
불가능하다는 말에 대한 일관된 반응.
```
플레이어: "이건 불가능해요!"
앨런 마스크: "잠깐. 물리법칙에 위배돼?"
플레이어: "아뇨, 그건 아닌데..."
앨런 마스크: "그럼 가능해. 다음!"
```

### 4.6 항상 음료수를 들고 다님
(실제: 업무 중 다이어트 콕 애용)
- 비주얼: 앨런 마스크가 항상 한 손에 음료수 컵을 들고 있음
- 우주복을 입고 있어도 빨대로 음료를 마시는 장면

## 5. 상황별 대사 시스템 (재설정 — 대중 버전)

```
[일반 실패]
```

## 7. 원본 소스 주요 에피소드 요약

### SpaceX 탄생 (2002)
- 러시아에서 ICBM 로켓 3번 구매 시도 → 가격 과다로 실패
- 귀국 비행기에서 로켓 제작 비용 스프레드시트 계산 → "직접 만들자!"
- 교훈: "계획이 안 됐을 때가 최고의 기회"

### Falcon 1 (2006~2008)
- 3연속 실패 → 4번째 성공 → 민간 최초 궤도 진입
- 4차 성공 직후 NASA 16억 달러 계약
- 교훈: "끈기가 로켓 연료보다 중요하다"

### Falcon 9 드론십 착륙 (2016)
- 수차례 해상 착륙 실패(폭발) → 성공
- 드론십 이름: "Of Course I Still Love You"
- 교훈: "실패 모음 영상을 자랑스럽게 공개하는 자세"

### Falcon Heavy + Starman (2018)
- 자기 차를 우주로 쏨 + "DON'T PANIC" + David Bowie BGM
- 두 부스터 동시 착륙
- 교훈: "꿈은 크게, 유머는 잃지 않게"

### Crew Dragon Demo-2 (2020)
