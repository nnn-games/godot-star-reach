# Godot 엔진 설치 튜토리얼 (2026년 기준 최신 버전 가이드)

게임 개발을 시작하기 위한 첫 번째 단계는 엔진을 설치하는 것입니다. 2026년 현재 (Godot 4.6 및 그 이후 버전 기준), Godot 엔진은 기존의 무거운 상용 엔진들과 달리 **"설치 과정이 필요 없는(Zero-install)"** 매우 가볍고 직관적인 구조를 지니고 있습니다.

이 문서는 가장 원활하게 Godot을 도입하고 C# 환경까지 설정할 수 있는 가이드를 제공합니다.

---

## 1. 내게 맞는 버전 선택하기 (Standard vs .NET)

Godot 엔진은 공식 홈페이지(https://godotengine.org)에서 두 가지 버전으로 제공됩니다. 본인의 주력 프로그래밍 언어에 따라 올바른 버전을 선택해야 합니다.

* **Standard 버전**: GDScript (Godot 전용 파이썬 형태의 스크립트 언어)와 C++(GDExtension)만 사용하는 경우 선택합니다. 엔진 파일 용량이 매우 작습니다. (보통 50MB 다운로드, 압축 해제 시 약 150MB 내외)
* **.NET 버전 (C# 지원)**: **C# (C-Sharp)** 언어를 사용하여 게임 로직을 작성하고 싶다면 반드시 이 버전을 다운로드해야 합니다. C# 구동을 위한 모듈이 포함되어 용량이 조금 더 큽니다.

> 💡 처음 Godot을 접하시고 튜토리얼을 따라 하실 예정이시라면 정보가 가장 많은 **Standard 버전(GDScript)** 사용을 강력히 권장합니다.

---

## 2. 엔진 다운로드 및 실행 방법 (공식 홈페이지 활용)

Godot 엔진의 공식 설치 방식은 놀랍게도 **'압축을 풀면 끝'**입니다. 복잡한 레지스트리 등록이나 수 기가바이트의 설치 과정이 존재하지 않습니다.

1. **공식 웹사이트 접속**: [Godot 다운로드 페이지 (godotengine.org/download)](https://godotengine.org/download)에 접속합니다.
2. **운영체제 확인**: 웹사이트가 접속자의 OS(Windows, macOS, Linux)를 자동으로 감지합니다.
3. **버전 다운로드**: `Godot Engine 4.X (현재 최신 Stable 버전, 예: 4.6)` 버튼을 클릭하여 ZIP 파일을 다운로드합니다.
4. **압축 해제**: 다운로드한 `.zip` 파일의 압축을 원하시는 폴더(예: `C:\Godot`)에 풉니다.
5. **실행**: 압축을 푼 폴더 안에 있는 `Godot_v4.6-stable_win64.exe` 와 같은 실행 파일을 더블클릭하면 엔진 런처(프로젝트 관리자)가 즉시 실행됩니다. 완료!

> 💡 엔진 관리 및 바로가기 생성: 실행 파일에서 우클릭하여 "작업 표시줄에 고정"을 해두시면 향후 편리하게 켤 수 있습니다.

---

## 3. 스토어를 통한 플랫폼 자동 설치 (옵션)

압축을 풀고 관리하는 것이 귀찮으시거나 자동 업데이트를 받고 싶으시다면, 기존에 주로 사용하시던 게임 플랫폼 런처를 통해서도 설치가 가능합니다. (이 역시 완전 무료입니다.)

* **Steam (스팀)**: Steam 상점 검색창에 "Godot Engine"을 검색하여 라이브러리에 추가 후 설치합니다. 
* **Epic Games Store**: 에픽게임즈 스토어에서도 Godot 검색 및 설치가 가능합니다. 자동 업데이트가 지원됩니다.
* **주의사항**: 위 스토어에서 직접 설치할 시 **기본적으로 Standard 버전**이 설치됩니다. C#(.NET) 환경을 반드시 사용해야 한다면 2번의 공식 홈페이지로 돌아가 수동 다운로드하시는 것이 가장 안전합니다.

---

## 4. C# 사용자를 위한 추가 설정 가이드 (.NET SDK 설치)

만약 1번 단계에서 `.NET 버전`을 다운로드하셨다면 추가로 Microsoft의 `.NET SDK` 설치가 필요합니다. 2026년 현재 최신 Godot 4.6 이상 버전에서는 주로 최신 .NET LTS (Long Term Support) 컴파일러(예: .NET 8.0 이상)를 요구합니다.

1. **.NET SDK 다운로드**: [Microsoft .NET 다운로드 페이지](https://dotnet.microsoft.com/download)로 접속합니다.
2. **최신 SDK 설치**: 화면에 보이는 권장 최신 SDK 버전을 다운로드하여 설치 프로그램을 실행합니다. (Runtime만이 아닌 반드시 **SDK**를 받으셔야 합니다.)
3. **재부팅 또는 환경변수 적용**: 설치가 완료되면 명령 프롬프트(cmd)를 열고 `dotnet --version`을 쳐서 버전 번호가 정상적으로 출력되는지 확인합니다.
4. **C# 편집기 준비**: Godot 엔진 내부에는 C#용 자동 완성 편집기가 분리되어 있으므로 **Visual Studio Code**, **Visual Studio 2022/2026**, **JetBrains Rider** 등 외부 편집기와 연동하여 사용하는 것을 추천합니다. `[편집기 메뉴 -> 에디터 설정 -> DotNet -> 외부 편집기]`에서 지정할 수 있습니다.

이제 새로운 프로젝트를 생성하여 Godot 생태계에서 즐거운 게임 개발을 시작하실 준비가 모두 끝났습니다!
